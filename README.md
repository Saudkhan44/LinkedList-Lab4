
# 📌 Linked List Menu Program in Java

## 🔹 Overview

This project is a **menu-driven program in Java** that demonstrates the complete implementation of **Linked Lists** with all important operations.
It supports:

* **Singly Linked List**
* **Doubly Linked List**
* **Circular Linked List**

The program provides a **console menu** to choose the linked list type and then perform various operations like **insertion, deletion, traversal, searching**, etc.
It is designed to help students and developers **learn Linked List concepts in Data Structures** step by step — from scratch to advanced.

---

## 🔹 Features

### 1. **Singly Linked List**

* Traversal (Iterative & Recursive)
* Insert at Beginning
* Insert at End
* Insert at Position (Middle)
* Insert After a Node
* Delete from Beginning
* Delete from End
* Delete at Position
* Delete by Value
* Search Element

---

### 2. **Doubly Linked List**

* Traversal (Forward & Backward)
* Insert at Beginning
* Insert at End
* Insert at Position
* Delete from Beginning
* Delete from End
* Delete at Position
* Search Element

---

### 3. **Circular Linked List**

* Traversal (Iterative & Recursive)
* Insert at Beginning
* Insert at End
* Insert at Position
* Delete from Beginning
* Delete from End
* Delete at Position
* Search Element

## 🔹 How to Run

1. Clone or download the project.
2. Open the project in **IntelliJ IDEA**, **Eclipse**, or any Java IDE.
3. Compile and run the main file:

```bash
javac LinkedListMenu.java
java LinkedListMenu
```

4. The console will display a menu:

```
====== Linked List Menu ======
1. Singly Linked List
2. Doubly Linked List
3. Circular Linked List
4. Exit
Enter your choice:
```

5. Select the linked list type and perform operations.
